# Sources

A source defines what gets backed up in a job. BackupOS supports eight source types, configured per-job on the new-job form.

<SourceTypes />

<Note>Each source type has optional parameters — include/exclude paths, database credentials, Proxmox API tokens — configured per-job. Credentials are encrypted at rest using the instance encryption key.</Note>
