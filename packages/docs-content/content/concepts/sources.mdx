# Sources

A source defines what gets backed up in a job. BackupOS supports:

| Source type | Notes |
|---|---|
| Filesystem | Any path accessible to the agent |
| Docker volume | Named Docker volumes |
| Docker container | App-consistent: pauses container, backs up volume, resumes |
| PostgreSQL | pg_dump before backup, hook-based |
| MySQL / MariaDB | mysqldump before backup |
| Redis | BGSAVE before backup |
| SQLite | File-copy with WAL checkpoint |
| Proxmox VM | Via Proxmox API, no in-guest agent needed |
| Proxmox LXC | Via Proxmox API |
| Windows system (VSS) | Volume Shadow Copy for consistent Windows backups |

Each source type has optional parameters (include/exclude paths, credentials for databases, etc.) configured in the job wizard.
