# Configure alerts

Go to **Settings → Alerts → New alert rule**. Choose a trigger:

<AlertTriggers />

<Tip>Start with "Job failed" and "No recent backup" — these two catch 90% of real incidents. Add storage threshold alerts once you know your typical growth rate.</Tip>

## Notification channels

- **Email** — configure SMTP in Settings → Notifications
- **Webhook** — POST JSON payload to any URL
- **Slack / Discord** — paste your webhook URL
