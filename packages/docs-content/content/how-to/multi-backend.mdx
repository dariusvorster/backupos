# Set up multi-backend replication

BackupOS supports multiple repositories per job. Each run writes to all assigned repositories in sequence.

1. Add a second repository (e.g. a local NAS alongside your R2 bucket)
2. Go to **Jobs → [job] → Repositories**
3. Click **Add repository** and select the second repo
4. Save

Both repositories receive the same snapshot on each run. If one repository is unavailable, the job continues with the other and flags a warning.
