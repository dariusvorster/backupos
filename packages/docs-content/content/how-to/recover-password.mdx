# Recover a lost repository password

If escrow was enabled when the repository was added:

1. Go to **Settings**. The escrow recovery section lists every repository with escrow enabled.
2. Select the repository
3. Enter your **recovery passphrase** (set when escrow was enabled)
4. BackupOS decrypts and displays the repository password

If escrow was not enabled and the password is lost, the backup data is permanently inaccessible. This is why escrow is strongly recommended for all repositories.
