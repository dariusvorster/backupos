# Restore an entire host

1. Go to **DR Mode** (or press the DR Mode shortcut: ⌘⇧D / Ctrl+Shift+D)
2. Select the restore spec for the host
3. Follow the guided checklist — each step shows instructions and a **Mark complete** button
4. Steps that can be automated (file restore, hook execution) run automatically

For a step-by-step guide on creating the restore spec in advance, see [Restore specs](/docs/concepts/restore-specs).
