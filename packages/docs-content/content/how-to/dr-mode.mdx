# Use DR Mode during an incident

DR Mode is a full-screen recovery checklist designed for use when calm thinking is hardest.

## Activate

- Press **⌘⇧D** (Mac) or **Ctrl+Shift+D** (Windows/Linux)
- Or: click the sun icon in the topbar

## What you see

Three recovery cards:
1. **Restore file** — quick single-file restore
2. **Restore database** — database dump restore
3. **Run restore spec** — guided full-host recovery

Select the appropriate card. For a full-host recovery, pick the restore spec for the affected system. Steps run in order; automated steps execute with one click.
