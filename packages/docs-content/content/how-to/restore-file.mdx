# Restore a single file

1. Go to **Snapshots**, find the snapshot
2. Click **Restore → Restore file**
3. Browse the file tree, select the file
4. Set destination path on the target agent
5. Click **Start restore**

The restore job runs like a backup job. Check the run log to confirm the file was written to the destination.
