# Back up multiple hosts on one schedule

Create a **job template** with the shared source type, repository, and schedule. Then create one job per agent from the template. Each job gets the same config but runs on a different agent.

Templates are in **Jobs → Templates → New template**. Apply a template when creating a new job by selecting it from the template dropdown.
