# Rotate API tokens

1. Go to **Settings → Security → API tokens**
2. Click **New token**
3. Set a name and scope (read-only or read-write)
4. Copy the token — it is shown only once
5. Update any clients using the old token
6. Revoke the old token: **⋯ → Revoke**

Rotate tokens at least every 90 days. Treat API tokens like passwords — never commit them to version control.
