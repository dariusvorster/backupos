# Connect a repository

A repository is where backup data is stored. BackupOS uses [Restic](https://restic.net) as its storage engine — all repositories are standard Restic repositories.

## Recommended: Cloudflare R2

Cloudflare R2 has zero egress fees, making it the cheapest option for most homelab setups. You pay only for storage (currently $0.015/GB/month).

### 1. Create an R2 bucket

1. Log into the Cloudflare dashboard
2. Go to **R2 → Create bucket**, name it (e.g. `homelab-backups`), choose a region
3. Go to **R2 → Manage R2 API tokens → Create API token**
4. Scope: **Object Read & Write** on your bucket
5. Note the **Access Key ID** and **Secret Access Key**

<Warning>Store the encryption password somewhere safe before clicking Initialise. If it is lost, the backup data is permanently inaccessible — Restic has no password recovery.</Warning>

### 3. Enable escrow (strongly recommended)

BackupOS can encrypt the password with your master passphrase and store it. You can recover it later via **Settings → Recover repository password**.

Click **Initialise repository**. BackupOS calls `restic init` and stores the connection details.
