# Security

BackupOS is designed with security as a first-class concern. All backup data is encrypted client-side before leaving the agent. The audit log is tamper-evident. TOTP is supported for all accounts.

This section covers: encryption model, TOTP enrolment, key escrow, audit log verification, API token scopes, and compliance considerations.

This section is being expanded. For TOTP setup, see [Settings → Security](/settings/security).
