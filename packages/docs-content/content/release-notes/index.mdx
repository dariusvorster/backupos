# Release notes

## Unreleased — V1 launch candidate

The V1 launch is in active preparation. Highlights of recent work:

- OIDC SSO with admin config UI (Authentik, Okta, Duo)
- Local password + TOTP 2FA via better-auth
- Password reset flow with email-based recovery
- Repository password escrow with Argon2id key derivation
- Tamper-evident audit log: SHA-256 hash chain across all privileged events
- Forensic mode on the audit page for investigating suspicious activity
- RBAC v1 — viewer role gating across destructive actions
- Verification edit/delete UI
- Datastore management UI for PBS-protocol backups
- Read-only integration API at `/api/v1/integration/*` (Bearer auth, scopes, rate limiting)
- Encryption-at-rest for stored credentials, with key rotation tooling
- Native systemd installer with self-update on `update --source`
- Per-channel UX for all 9 alert channels (email, Slack, Discord, Telegram, Pushover, Gotify, ntfy, generic webhook, Twilio SMS)
- Grid Shield logo and dynamic favicon (idle / running / alert states)

A full changelog will be published with V1 GA.
