Snapshot picker — how does it trigger?

When you want to restore from a specific snapshot, how do you get to the picker?

A

Modal on "Run now"

"Run now" always opens a modal first. Choose a snapshot (or keep "latest"), then confirm. One button, one flow.

B

Split button

"Run now" fires immediately with latest. The ▾ dropdown on the right reveals "Choose snapshot…" for a picker. Fast path stays fast.

Run now ▾
Run with latest
Choose snapshot…
C

Separate "Run with snapshot…" button

"Run now" stays as-is (runs latest). A secondary ghost button opens the picker. Two explicit buttons, zero ambiguity.
